
## toolBarItem:注册顶栏按钮

中文:注册顶栏按钮

英文名:registToolbarItem

### 功能说明

在窗口顶栏增加一个按钮

### 参数

```js
[
  {
    "位置": "控制按钮出现在窗口的哪一侧"
  },
  {
    "配置": {
      "文字或者label": "按钮的文字提示",
      "图标或icon": "按钮显示的图标,可以通过注册图标接口添加可用的图标",
      "点击回调函数或者callback": "点击按钮时的回调",
      "事件配置或者events": "不同事件的回调函数,当提供此项时,点击回调函数项无效",
      "元素或者element": "直接传入按钮内容元素,不推荐使用"
    }
  }
]
```

### 返回值

没有返回值

### 其他

没有返回值
