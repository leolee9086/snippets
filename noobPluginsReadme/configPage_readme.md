
## configPage:打开网页页签

中文:打开网页页签

英文名:openPage

### 功能说明

打开一个网页页签,常用于打开设置页签或者打开文件之类

### 参数

```js
{
  "地址": "页签所在的地址",
  "关闭回调或callback": "页签关闭时的回调函数"
}
```

### 返回值

没有返回值

### 其他

没有返回值
