## 安装方式

使用noob-core包可以在集市模块直接安装它。

当然你也可以通过其他方式进行安装，例如直接下载包原始文件。

## 使用方法

### 基本使用

在开启插件之后，打开了头图的文档会多出一些按钮，点击这些按钮就可以加载对应来源的图片了。

### 进阶使用

头图按钮是可配置的，观察它们的结构，你也可以尝试写一下你自己的头图按钮。

## 依赖

这个插件可以不依赖noob的后端模块,在纯前端模式下安装和使用,但是需要一定的修改。

但是建议还是配合noob-core的完整版本来使用它，因为作者是在electron环境下开发的这个插件，如果离开noob，有可能某些功能无法完整使用。
